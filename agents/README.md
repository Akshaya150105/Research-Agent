# 📘 README — Critic Agent & Gap Detector Agent

## 🚀 Overview

This project contains two AI agents for research paper analysis:

### 🔍 Critic Agent

Analyzes a single paper and identifies:

- Weaknesses  
- Missing evaluations  
- Poor baselines  
- Methodological issues  

👉 **Output:** Enriched JSON  

---

### 🧠 Gap Detector Agent

Analyzes one or more papers to find:

- Research gaps  
- Missing method–dataset combinations  
- Repeated unresolved limitations  
- Cross-paper contradictions  

👉 **Output:** Gap report JSON  

---

## 🧠 Pipeline Flow

```
Raw Paper JSON
   ↓
Critic Agent
   ↓
Enriched JSON
   ↓
Comparator
   ↓
Gap Matrix
   ↓
Gap Detector
   ↓
Final Gap Report
```

## 🧪 1. Critic Agent

### 📌 What it does

Reads a research paper JSON and detects:

- Weak experiments  
- Missing comparisons  
- Poor generalization  

Adds:

- `critiques`  
- `critique_summary`  

---

### ▶️ Run Command

```bash
python agents/critic_agent.py data_1\parsed\paper.json --verbose
```

#### With LLM:

```bash
python agents/critic_agent.py data_1\parsed\paper.json --ollama-host <enter your url> --verbose
```

---

### 📤 Output

Saved as:

```
data_1\parsed\<paper>_enriched.json
```

---

## 🔍 2. Gap Detector Agent

### 📌 What it does

Uses 3 strategies:

#### 🧩 Combinatorial
- Uses `gap_matrix.json`
- Finds missing method × dataset combinations  

#### ⚠️ Limitation-Based
- Uses ChromaDB semantic clustering  
- Finds repeated unsolved problems  

#### 🔄 Cross-Paper
- Uses comparator outputs  
- Finds contradictions between papers  

---

### ▶️ Run Command

```bash
python agents/gap_detector_agent.py data_1\parsed\1706.03762v7\attention_is_all_you_need_2023_enriched.json --ollama-host <enter your url> --verbose
```

---

### ▶️ Multiple Papers Example

```bash
python agents/gap_detector_agent.py ^
 data_1\parsed\paper1_enriched.json ^
 data_1\parsed\paper2_enriched.json ^
 --verbose
```

---

### ▶️ Without LLM

```bash
python agents/gap_detector_agent.py data_1\parsed\paper.json --no-llm --verbose
```

## 📦 Prerequisites for Gap Detector

### ✅ Required

1. **Enriched JSON (from Critic Agent)**  
```
data_1\parsed\*_enriched.json
```

2. **Gap Matrix**  
```
memory\gap_matrix.json
```
Generated by: `enricher.py`  
Prevents combinatorial explosion  

3. **Comparator Outputs**  
```
data_1\agent_outputs\comparisons\*.json
```
Enables cross-paper gap detection  

4. **ChromaDB Store**  
```
rag\chroma_store\
```
Used for semantic limitation clustering  
If missing → fallback to Jaccard
